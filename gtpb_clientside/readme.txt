This project is the user interface layer for agileBase.

It consists of Velocity templates, Javascript and CSS.

For information on the templating language, see
http://velocity.apache.org/

== Where things are ==

The main root of UI templates is WEB-INF/templates/gui

-- Pane 3 --
Locations of important files for pane 3:

jQuery and libraries are in /resources

Javascript and CSS to manage the tabs is in /resources/tabs

CSS for report fields and manage tabs is the same but currently duplicated in files
/resources/tabs/report_fields.css
/resources/tabs/report_administration.css
